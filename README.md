## About the Project

The **Quiz Game (Quizzler App)**   is an interactive Python quiz application that fetches real-time questions from the Open Trivia Database API.
Players answer True/False questions and the score increases with every correct answer.

This project strengthens understanding of:
- Object-Oriented Programming
- Building GUIs using Tkinter
- Working with REST APIs
- Game logic and feedback handling
- Dynamic UI updates
The application uses a clean and attractive interface, automatically loads new questions, and disables buttons after the quiz is complete.
---

## Key Features
- Fetches live quiz questions using API
- Clean and interactive Tkinter UI
- Score updates in real time
- Automatic feedback with color indicators (green/red)
- Tracks question number and progress
- End-of-quiz screen with disabled buttons
- Uses HTML unescaping to display clean questions
